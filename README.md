# 001、需求
>后台实时作业、周期性任务=》根据统计条件，统计指定页面专题结果，生成图片并保存。
# 002、手动场景
>登录平台，打开指定页面，在【统计条件】部分选择检索条件，点击【统计】按钮，等待专题结果加载完成。
# 003、模拟场景
>打开指定页面，根据统计条件，统计指定页面专题结果，生成图片并保存到服务器。
# 004、领域模型
>查看项目目录下snapshot_domainModel.png、snapshot_domainModel.oom文件。
